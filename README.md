

Modular unstructured mesh based hydrological model.
![](https://github.com/Chrismarsh/CHM/blob/master/mesh.png)

#Dependencies
* Matlab (optional)
* Armadillo
* boost
* GNU GSL
* Intel TBB
* CGAL
* Paraview (if building the filter) otherwise VTK
* C++11 compliant compiler

#To build:
    cmake .
    make

#To test:
    cmake .
    make check
    make test
